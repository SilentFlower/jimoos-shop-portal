# 积墨开源商城 后台管理

[积墨开源商城](https://github.com/jimoos-cn/jimoos-shop-server) 配合使用的 商城后台管理。

使用技术栈 为 vue2 + vuex + [Ant Design Pro of Vue](/pro.antdv.com)

## 本地开发

> yarn install

> yarn run serve

启动完成后会自动打开浏览器访问 http://localhost:8000
你看到登录页面就代表成功了。
